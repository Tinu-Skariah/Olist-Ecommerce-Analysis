# Olist-Ecommerce-Analysis
The E-Commerce Performance Insights Dashboard analyzes sales, customer behavior, payments, and delivery performance using the Olist dataset. It highlights key metrics, sales trends, top cities, payment preferences, and the impact of shipping time on reviews, with interactive filters for deeper analysis.
